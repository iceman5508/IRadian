subString()

subString(  $action = 'first', integer  $numberOfChars) : string

Get part of the string

Parameters

$action
  • options are first and last first, set pointer to the start of the string. last, set pointer to the end of the string However action can also be an integer value of where in the string you wish to start
integer $numberOfChars
  • number of chars to include. if pointer is first, the char count will be from left to right. if point is last, the char count will be from right to left.

Returns

string

Tags

note

this method ignores white spaces
